# MyGPA Backend API

Express.js backend for the MyGPA application.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/mygpa
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
```

## Running

```bash
# Development (with auto-reload)
npm run dev

# Production
npm start
```
